# Student Record Management System (SRMS)

## Description

A simple Python program that collects student information, stores it in a Pandas DataFrame, and exports the data to an Excel file (`Student_record.xlsx`).

## Features

* Add records for multiple students
* Store Name, Age, Course, Gender, Roll Number, and Address
* Display records in a table
* Export data to an Excel file

## Requirements

* Python 3
* Pandas
* OpenPyXL

Install the required libraries:

```bash
pip install pandas openpyxl
```

## How to Run

1. Run the Python script.
2. Enter the number of students.
3. Fill in the required details.
4. The records will be displayed and saved as `Student_record.xlsx`.
