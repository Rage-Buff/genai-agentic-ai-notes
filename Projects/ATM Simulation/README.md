# 🏦 Python ATM Simulation Using OOP (Encapsulation)

## README

This project is a simple **ATM Simulation System** built in Python using **Object-Oriented Programming (OOP)** concepts, with a focus on **Encapsulation**.

### Features

*  Secure PIN verification using private methods
*  Check account balance
*  Withdraw money
*  Change ATM PIN
*  Interactive menu-driven interface

### OOP Concepts Used

* **Class & Object**
* **Constructor (`__init__`)**
* **Methods**
* **Encapsulation**

  * Private attributes (`__pin`, `__balance`)
  * Private method (`__ask_pin()`)

### How to Run

1. Save the code as `atm.py`.
2. Run the program:

   ```bash
   python atm.py
   ```
3. Use the menu to perform ATM operations.

**Default PIN:** `0000`
**Initial Balance:** `₹2000`
